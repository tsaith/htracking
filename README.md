# Hand tracking via computer vision

Python 3 is expected.

## Packages required

* [Numpy](http://www.numpy.org/)
* [Ipython](http://ipython.org/)
* [TensorFlow](https://www.tensorflow.org/)
* [Keras](http://keras.io/)
* [matplotlib](http://matplotlib.org/)
* [OpenCV](https://opencv.org/)

## Version information

Python 3.6
Tensorflow: 1.4.0
Keras: 2.1.5
Pytorch: 0.4.0
OpenCV: 3.4.1
